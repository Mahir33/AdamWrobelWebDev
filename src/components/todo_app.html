The Smart ToDo

HTML5, CSS3, JS App

With this classic ToDo application, the user can easily manage daily tasks. The application allows to add new, or edit and / or delete existing tasks.

  • Link: ToDo App
  • Link: GitHub source code

How to use:

  1. Enter the name of the todo.
  2. Click "add" button.
  3. If the task is completed, click the blue button "completed".
  4. If you want to edit todo, click "EDIT" button.
    • Edit existing todo text, then click the "CONFIRM" button.
    • If you want to cancel the changes, click the "CANCEL" button.
  5. If you want to delete todo, click the red button "delete".
