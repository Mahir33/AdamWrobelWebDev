All road leads to Plannerio

CSS3, SASS, JS, API, React.js

Plannerio is a Social Media App that allows you to share photos with other users and plan a trip to where they were taken.

  • Link: GitHub source code

  • CSS3: SASS / BEM

  • React.js
    • component design
    • component development

    • some of a functionality
