The Weather Check

HTML5, CSS3, JS, API App

With this application, the user can check the weather all over the world with just one click - you can get information about the weather conditions, temperature and humidity in a specific city on earth.

  • Link: Weather App
  • Link: GitHub source code

To make the application, I used the remote API from https://openweathermap.org/. Pictures that show weather conditions have been included in the conditional instructions and are dependent on the id downloaded from the API - thanks to this solution, the pictures change depending on the current weather conditions in a given region on earth.

How to use:

  1. Enter the name of the city you are interested in in the appropriate field.
  2. Use the left mouse button to click the "Check" button or use "Enter" on your keyboard.
  3. The weather data will be displayed in the bottom panel of the app.
